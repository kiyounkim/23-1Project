## 🔎About the game🔎
1. 우주 배경 3D TPS 게임. 
2. WASD와 스페이스바, Left Shift를 이용하여 우주선을 조종할 수 있다.
3. 목표는 오래 생존하기와 점수를 높게 얻기.
4. 마우스 좌클릭을 통해 미사일 발사 가능.
5. Left Shift 를 통해 제자리 정지 가능
